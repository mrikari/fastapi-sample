# ことはじめ

[uvicorn-gunicorn-fastapi-docker](https://github.com/tiangolo/uvicorn-gunicorn-fastapi-docker) を参考にまずは起動。

```bash
docker-compose build --no-cache
docker-compose up -d
```

うまく起動したら早速アクセス: [http://localhost:8000/](http://localhost:8000/)
